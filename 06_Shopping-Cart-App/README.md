# Shopping Cart App

## Tech Used

- Vite
- React 19
- Redux
- JSON Server
- Bootstrap 5
- Bootstrap icons
- CSS

##  Installation

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn or pnpm

### Clone Repository

```bash
# Clone the repository
git clone https://github.com/GourangaDasSamrat/react-labs

# Navigate to project directory
cd 06_Shopping-Cart-App
```

### Install Dependencies

```bash
# Using npm
npm install

# Or using yarn
yarn install

# Or using pnpm
pnpm install
```

### Development Server

```bash
# Start development server
npm run dev

# Application will be available at http://localhost:5173

```
```bash
# Start development server
npm run db

# Mock databse will be available at http://localhost:4000
```

### Build for Production

```bash
# Create production build
npm run build

# Preview production build
npm run preview
```
